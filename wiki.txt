# Litecode Language Features

## Display Command
The `display` command is used to print text or integers to the console.

### Syntax
```
display("text")
display('text')
display(integer)
```

### Examples
```
display("Hello, World!")
display('Hello, World!')
display(123)
```

## Variable Declaration
Variables can be declared and assigned values. The type of the variable is automatically determined based on the value assigned. Strings must be enclosed in double quotes (`"`) or single quotes (`'`).

### Syntax
```
variable = value
```

### Examples
```
name = "John Doe"
age = 30
greeting = 'Hello, World!'
```

### Notes
- Variable names must not contain spaces.
- Strings must be enclosed in matching quotes.
- Integers are automatically detected if they are not enclosed in quotes.
